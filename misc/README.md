# Misc – Diagrams & Documentation

This folder contains visual documentation for the ACME TeamHub project:

| File | Contents |
|------|----------|
| [architecture.md](./architecture.md) | AWS cloud architecture & local development setup |
| [data-flows.md](./data-flows.md) | API request flows, auth flow, CRUD lifecycle, ER diagram |
| [pipelines.md](./pipelines.md) | Deployment pipelines, service discovery, Terraform IaC flow |
| [user-stories.md](./user-stories.md) | 8 user story diagrams with sequence/state/flowchart diagrams |

All diagrams use [Mermaid](https://mermaid.js.org/) syntax and render in GitHub, VS Code (with Mermaid extension), and most markdown viewers.
